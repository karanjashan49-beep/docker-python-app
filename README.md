# 🐳 Docker Python App

## 📌 About

This project demonstrates how to containerize a Python application using Docker.

The goal of this project is to understand the fundamentals of Docker, including images, containers, Dockerfiles, and application packaging.

---

## 🚀 Features

- Simple Python application
- Dockerized environment
- Portable execution
- Easy deployment

---


## ▶️ Running the Application

Run locally:

```bash
python3 app.py
```

Build Docker image:

```bash
docker build -t docker-python-app .
```

Run container:

```bash
docker run docker-python-app
```
